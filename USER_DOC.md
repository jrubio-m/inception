
* ### _.env_

The minimum variables that the file must contain:

```bash
DOMAIN_NAME=<login>.42.fr

DB_NAME=
DB_USER=
DB_HOST=

WP_TITTLE=Inception
WP_ADMIN_USER=
WP_ADMIN_EMAIL=
WP_USER=
WP_USER_EMAIL=
```
* ### _secrets_

The password files you will need:

- db_password.txt
- db_root_password.txt
- wp_password.txt
- wp_admin_password.txt
